        How the footprint is computed

        Given a base model, we build the lineage DAG (parent → child) and take the base plus all descendants as the family. Each model reports only its incremental training delta; the family total sums the subtree with deduplication for merged nodes. Coverage shows what fraction disclosed a dia_report — totals are a lower bound when reports are missing.

        Graph interaction inspired by Model Atlas (Horwitz et al., 2025); footprint data is DIA-only.
